# Node adb scrcpy

Node adb scrcpy 是一个基于 Node.js 的 ADB 命令行工具，可以方便的启动 Scrcpy。

![Alt text](image.png)

## 先决条件

- 请确保 ADB 和 Scrcpy 是在可用状态
- 可在终端下使用 adb 和 scrcpy -h 命令检查是否可用
- 执行 ``` adb ``` 命令后
  
  ![Alt text](image-1.png)
- 执行 ``` scrcpy -h ``` 后

- ![Alt text](image-2.png)
